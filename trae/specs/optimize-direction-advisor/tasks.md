# Rollback Strategy

如发现重大问题:
1. 撤销 `references/philosophy.md` 改动 → git revert 整个 commit
2. 撤销 `references/implementation.md` 改动 → 恢复 §4 设计建议报告模板的旧版
3. 撤销 `SKILL.md` 改动 → 恢复 §1.4 + 导航表 + 版本号
4. 删除 `.trae/specs/optimize-direction-advisor/` 目录

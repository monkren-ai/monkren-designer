# 03 · 设计执行（Design Execution）

> monkren 设计生命周期的**第 3 阶段**。把方向变成现实——高保真 / 可交互的产出物。

## 本阶段 skill

| Skill | 状态 | 用途 |
|-------|------|------|
| [make-a-prototype](./make-a-prototype/) | ✅ 活跃 | 输出可交互 HTML 原型（hi-fi） |

## 阶段问什么

- **怎么做出来**？——把 wireframe/选定方向变成真实可见的成品

## 核心方法

- **直接出可交互的 HTML**（不是 Figma 截图，不是静态设计稿）
- 5 维度评审中的「细节执行」从这阶段开始严格
- 输出后必须能跑（能点 / 能动 / 能看）

## 阶段产物

- 1 个可交互的 HTML demo
- 链接、文件路径、可访问的预览
- 5 维度自评（v6.0 新增：执行阶段交付前做一次自评）

# 02 · 创作定义（Creation Definition）

> monkren 设计生命周期的**第 2 阶段**。决定做什么、探索多个方向、选一个——做设计前的"选方向"。

## 本阶段 skill

| Skill | 状态 | 用途 |
|-------|------|------|
| [generate-variations](./generate-variations/) | ✅ 活跃 | 同一设计产出 3+ 风格方向 |
| [wireframe](./wireframe/) | ✅ 活跃 | 低保真线框探索布局方案 |

## 阶段问什么

- **做什么**？——layout / 信息架构 / 流程
- **选哪个**？——在 3+ 方向里挑 1 个

## 核心方法

- **不要直接做 hi-fi**——先 lo-fi（wireframe）+ 风格方向（variations）
- 用最便宜的方式探索最多的可能性
- 5 维度评审中的「创新性」从这阶段开始打分

## 阶段产物

- `wireframe` 输出：3-5 个布局变体（纯黑白线框）
- `generate-variations` 输出：3+ 风格方向（每个方向 1-2 张图）
- 用户选择 1 个方向 → 进入 03 设计执行

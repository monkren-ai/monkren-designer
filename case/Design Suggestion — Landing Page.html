Current State

当前 v6.0 landing page 采用 Graphic Monochrome Canvas 2.0 哲学：纯黑/纯白/单色灰 + 0 圆角 UI（仅 .mark 50% 例外） + 1px 边框 + 3 字体职责分明 + DEPTH meter 滚动驱动视觉签名。1+3+1 结构（Hero / Proof / Method / Disciplines / Difference 5 个 section）极简克制。3 case 卡片是页面最"通用"区块——hover 翻色模式未与 discipline / stage 的视觉语言统一。hero 4 query 漏 03 设计执行阶段，hero compat (5) 与 disciplines section (7) 数量不一致，8 处硬编码值未 token 化。整体 7.0/10，已具独特性但执行细节可精进。

Directions

★ Recommended
Monochrome Hardening
深化 v6.0 现状
  • 把 DEPTH meter 从装饰升级为 scrub 控件——hover 时显示"← 滚回顶部"affordance
  • 3 case 卡片从通用 hover 翻色改为左侧 2px --color-warn 边框 + num 角标，与 .discipline / .stage 视觉语言统一
  • --color-warn 用法收窄：仅 1 个 query 边框 + discipline num + stage num 3 处，删除多余（强化"唯一强调色"原则）
Quick Win：hover case card 显示 warn left border → 5 min
#2 Alternative
Monochrome + Slate
扩展设计语言

#3 Bold Choice
Monochrome + Texture
换基础范式
  • body 背景加 SVG noise texture（feTurbulence 生成，0.04 opacity）——把"纯黑画布"从概念升为物理质感

  • Method section stage 卡片加 1px slate 底边作"渐进深度"暗示（与 DEPTH meter 呼应）
Quick Win：body 背景加 SVG noise (0.04 opacity) → 5 min

Recommendation

→ Monochrome Hardening
